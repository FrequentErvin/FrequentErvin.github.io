Interactive Problem Repository ♣

This is a random problem generator with a twist. I've seen things very similar to this, for example: Marabot, Trivial and AMC trainer. But these are mainly for AMC/AIME and pre-olympiad level math. I wanted to make something for IMO participants and people near that level. The most important aspect is the MOHS hardness which is determined by user feedback.

Mathematical Olympiad Checklist ☑

This is the mathematical analogue of the OI Checklist. With this you can mark and save the problems you've done from the major math competitions. This is intended for math olympians during training for the IMO and similar. The problems are linked so you can go and look up the solution if stuck.

Cow Gazer 🤠

A computer vision challenge that I'm currently working on with people from Isaac Code. The main task is to classify cows on a farm by using Convolutional Neural Networks. The main product will be outrolled as an Android app for the farmers in Livno to use.
