This is a project of mine which I've aspired to make and perfect for a long time now. So what is this? This is a random problem generator with a twist. I've seen things very similar to this, for example: Marabot, Trivial and AMC trainer. But these are only for AMC and pre-olympiad level math. I wanted to make something for IMO/RMM/APMO participants. I added a new aspect to it with: 1) Machine learning to find anomalies and spit out enjoyable problems (based on feedback from user) 2) MOHS hardness is completely determined by user feedback. The 2) part is crucial since all of the above mentioned generators don't have this interactive aspect, which in my honest opinion makes a huge difference in the enjoyability of the process.

Things that I'll add: timer, score, enjoyability of problems, hardness of problem, link to solution, solved, skip, show solution.
