Introduction - PyTorch

PyTorch is a popular Deep Learning framework based on the Torch library. Software in Tesla Autopilot, Uber's Pyro, Hugging Face's Transformers and more are implemented using this framework. It is growing in popularity as most papers that are published in the field of machine learning especially in computer vision and natural language processing have been tested and implemented using it. As to why it's so popular, two main arguments hold:

  1. Tensor computing with strong acceleration via GPUs
  2. Deep neural networks built on an automatic differentation system using computational graphs
These make PyTorch incredibly fast, versatile, and efficient. Hopefuly throughout this post we'll get a chance to experience those perks ourselves.

PyTorch tensors

A tensor can be thought of as an n-dimensional array. It isn't 100% mathematically correct, since we would be talking about ranks but for understanding purposes I stick with this. Vectors are 1D tensors while matrices are 2D tensors. This term dominates all implementations of machine learning applications. Usually, tensors store parameters which interact with other tensors using special operations defined in the field of linear algebra, which you can understand easily without prerequisite knowledge. PyTorch modules

The PyTorch ecosystem contains 3 high-level modules and many more submodules. Here they are:

  1. Autograd module

    2. The method that belongs to this module called automatic differentiation is key for the tuning of parameters in neural networks. This method computes the gradients of the output with respect to each parameter in the neural network. It does so with the use of computational graphs where it, roughly speaking, "remembers" the operations you've done throughout your network and then replays it back to compute the gradients.

  2. Optim module

    3. torch.optim is a module that implements optimization algorithms used for building NNs. This speeds up building NNs significantly.

  3. nn module

    4. The nn module provides layers and tools to easily create the neural network architecture by just defining the layers of the network.

PyTorch also has many useful submodules such as data loading utilities which we will use below.
